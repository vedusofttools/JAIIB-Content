# Core Banking Systems & Accounting in Computerised Environment

============================================================

## 1. Meaning of Computerised Accounting

Computerised Accounting means:

Recording, processing and reporting of financial transactions
Using computer systems and software.

Instead of manual ledgers,
Transactions are entered into software.

============================================================

## 2. Features of Computerised Accounting

- Real-time processing
- Automatic posting to ledger
- Faster report generation
- Reduced manual errors
- Centralised database
- Data backup and recovery

============================================================

## 3. Key Terms in Computerised Accounting

------------------------------------------------------------

Voucher:
Electronic entry of transaction.

Ledger:
Account maintained in system.

Database:
Central storage of financial data.

User Access Control:
Restricting system access to authorised users.

Audit Trail:
Record of all changes made in system.

============================================================

## 4. Difference Between Manual and Computerised Accounting

Manual Accounting:

- Handwritten books
- Time consuming
- Higher risk of calculation errors
- Difficult data retrieval

Computerised Accounting:

- Automated posting
- Faster processing
- Easy report generation
- Better internal control

============================================================

## 5. Advantages of Computerised Accounting

- Speed
- Accuracy
- Data security
- Easy reconciliation
- Multi-branch integration
- MIS reporting

============================================================

## 6. Disadvantages / Risks

- System failure risk
- Cyber security risk
- Data manipulation risk
- High setup cost
- Dependence on IT infrastructure

============================================================

# 7. Core Banking System (CBS)

Core Banking System means:

Centralised banking solution
Where all branches are connected to a single server.

Customer can operate account from any branch.

============================================================

## 8. Core Banking Components

- Central server
- Branch terminals
- Database management system
- Network connectivity
- Security systems
- Backup systems

============================================================

## 9. Scope of Computerisation in Banking

Computerisation covers:

- Deposit accounts
- Loan accounts
- ATM transactions
- RTGS/NEFT processing
- Clearing operations
- Interest calculation
- Regulatory reporting

============================================================

## 10. Information Security in Banking

Banks must ensure:

- Data confidentiality
- Data integrity
- Availability of systems
- Protection from cyber attacks
- Firewalls and encryption
- Multi-factor authentication

Cyber risk is major operational risk.

============================================================

## 11. Audit in Computerised Environment

Important controls:

- Maker-checker system
- Segregation of duties
- Access control
- Regular backups
- System logs review

Audit trail helps detect fraud.
